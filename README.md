# Bookshelf

A visual exploration of my reading data. This project transforms book statistics into bold, information-driven graphics inspired by brutalist and Swiss design principles.

**[View Live Site](https://eline00.github.io/bookshelf.github.io/)**

---

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
